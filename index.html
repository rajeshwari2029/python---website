Learn Python: Indentation, Syntax & Comments

Welcome

This website teaches you the basics of Python's indentation, syntax, and how to write comments.

Indentation in Python

Indentation is crucial in Python. It defines blocks of code and replaces braces used in other languages.

def greet(): print("Hello, World!") if True: print("Indented block")

Basic Python Syntax

Python is known for its clear and readable syntax. Statements end with a newline.

x = 10 y = 20 sum = x + y print(sum)

Comments in Python

Comments start with a # symbol and extend to the end of the line.

# This is a single-line comment print("Hello") # Inline comment ''' This is a multi-line comment (using triple quotes) '''
